
### masker-database

- 项目简介: 数据库SDK（包括多数据源切换、本地事务管理）

- 主要功能: 
   - 提供本地事务控制功能(MyBatis)
   - 提供本地事务控制功能(Spring)
   - 提供数据源动态切换功能(Spring)
   - 提供数据源动态切换功能(Spring+拦截器)

- 版本引用:

```xml
<dependency>
    <groupId>io.github.jiashunx.devkit</groupId>
    <artifactId>masker-database</artifactId>
    <version>x.y.z-SNAPSHOT</version>
</dependency>
```
